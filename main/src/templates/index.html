How the agent works

The Query Agent uses your Weaviate collections to return grounded responses and supports follow-up refinement.

1. Prepare data Collections are created and seeded from JSON.
2. Interpret prompt Your natural-language question is mapped to retrieval behavior.
3. Return answer Results are composed from products and brands into a concise response.
